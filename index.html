マイクラコマンド表

※ 統合版の場合、minecraft:が付いている場合も
minecraft:は必要ありません

コマンド 引数・セレクタ 説明 エディション コピー
/give @p minecraft:diamond 1 ダイヤモンドを1個付与
「1」の部分を変えればその個数
「diamond」の部分を変えれば、別のアイテムが指定した個数分もらえる		 統合版, Java版
/tp @a 100 64 200 全員を指定座標にテレポート 統合版, Java版
/effect @a speed 60 1 全員にスピード効果（60秒間、レベル1）
レベルを上げすぎると、とんでもない事になるので注意		 統合版, Java版
/summon minecraft:zombie ゾンビを召喚 統合版, Java版
/weather clear 天気を晴れに設定 統合版, Java版
/kill @e[type=creeper] すべてのクリーパーを倒す
type=の後を変えれば他のモブにも出来る		 統合版, Java版
/list サーバー上のプレイヤーを一覧表示 統合版, Java版
/locate village 最寄りの村の座標を表示
villageの場所を他の場所の名前に変更すると
他の場所の最寄りの座標が表示される。
ネザー要塞やエンドシティなどは、その世界(ディメンション)に居なければ実行不可		 統合版, Java版
/fill 0 64 0 10 64 10 minecraft:stone 指定範囲を石で埋める 統合版, Java版
/setblock 100 64 100 minecraft:gold_block 指定座標に金ブロックを設置 統合版, Java版
/time set day 時間を昼に設定 統合版, Java版
/gamemode @p creative プレイヤーのゲームモードをクリエイティブに変更 統合版, Java版
/difficulty peaceful 難易度をピースフルに設定 統合版, Java版
/spawnpoint @p 100 64 100 プレイヤーのリスポーン地点を設定 統合版, Java版
/setworldspawn 0 64 0 ワールドのスポーン地点を設定 統合版, Java版
/clear @p 近くのプレイヤーのインベントリを空にする
全てのユーザーの場合は@a
自分のみの場合は@s		 統合版, Java版
/enchant @p minecraft:sharpness 5 プレイヤーの持ち物に鋭さVを付与 統合版, Java版
/xp @p 100 プレイヤーに100レベルの経験値を付与 統合版, Java版
/title @a title ようこそ！ 全員にタイトルメッセージを表示 統合版, Java版
/playsound minecraft:entity.experience_orb.pickup master @a 全員に経験値取得音を再生 統合版, Java版
/say こんにちは、世界！ チャットにメッセージを表示 統合版, Java版
